# サークル情報データ管理ガイド

このディレクトリには、サークルの人数と受賞歴の件数を管理するデータファイルが含まれています。

## ファイル構成

- `circle-info.json` - サークルの人数と受賞歴データ
- `index.ts` - データアクセス関数

## データの更新方法

### メンバー数の更新

`circle-info.json` の `memberCount` セクションを編集します：

```json
{
  "memberCount": {
    "active": 25,           // 実際に活動しているメンバー数
    "discord": 100,         // Discordサーバーのメンバー数
    "lastUpdated": "2025-03-01"  // 最終更新日（YYYY-MM-DD形式）
  },
  "awardsCount": 3          // 受賞歴の件数
}
```

### 受賞歴の件数の更新

新しい受賞があった場合は、`awardsCount` の数値を増やしてください：

```json
{
  "awardsCount": 4  // 受賞が増えたら数値を更新
}
```

## 注意事項

- JSONの構文エラーに注意してください（カンマの付け忘れなど）
- 日付は `YYYY-MM-DD` 形式で記載してください
- データを更新後、開発サーバーを再起動するか、ページをリロードすると変更が反映されます

## 変更の確認

データを更新した後、以下のコマンドで動作確認できます：

```bash
# 開発サーバーを起動
npm run dev

# ブラウザで確認
# - http://localhost:3000 (ホームページの実績セクション)
# - http://localhost:3000/about (Aboutページの活動詳細セクション)
```

更新内容が両方のページに反映されていることを確認してください。
